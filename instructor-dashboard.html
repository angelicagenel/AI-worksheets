How to add to Canvas:
  1. Open Canvas → your Course → Assignments → + Assignment
  2. Set Submission Type → External Tool
  3. Click Find and select Spanish Learning Edge (or enter the URL manually)
  4. Paste the workbook URL above into the External Tool URL field
  5. Set points, due date, and attempts in Canvas as usual → Save

Canvas handles dates, attempt limits, and scoring automatically — you don't need to configure those in the workbook.
